
# Telegram TikTok & Facebook Downloader Bot

## Features
- Download TikTok / Facebook videos
- Download all TikTok username videos
- Download all Facebook Page videos
- Filter by views / likes / date
- Auto ZIP results
- Real-time progress
- Auto update (git pull)
- Auto restart
